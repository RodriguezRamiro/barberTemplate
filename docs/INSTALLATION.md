# Installation Guide

Thank you for purchasing Clipse Grooming.

This guide will walk you through setting up the template on your computer and preparing it for customization and deployment.

---

# Requirements

Clipse Grooming is a static HTML template and requires no frameworks, package managers, build tools, or dependencies.

You only need:

* A web browser
* A text editor or code editor

Recommended editors:

* Visual Studio Code
* Sublime Text
* WebStorm
* Cursor

---

# Step 1: Download The Template

After purchasing the template, download the ZIP archive from Gumroad.

Example:

Clipse-Grooming-v1.0.zip

---

# Step 2: Extract The Files

Extract the ZIP archive to a folder on your computer.

Example:

Desktop/
└── Clipse-Grooming-v1.0/

---

# Step 3: Open The Project Folder

Open the extracted folder.

You should see a structure similar to:

Clipse-Grooming-v1.0/
├── assets/
├── docs/
├── index.html
├── script.js
└── style.css

---

# Step 4: Launch The Website

Open:

index.html

inside your web browser.

The template will run immediately without additional setup.

No installation commands are required.

---

# Step 5: Open The Project In Your Editor

Open the project folder in your preferred editor.

Example with Visual Studio Code:

1. Open Visual Studio Code.
2. Select File → Open Folder.
3. Choose the Clipse Grooming project folder.

You can now begin customizing the template.

---

# Editing Content

Most business information can be updated directly inside:

index.html

Examples include:

* Business name
* Services
* Pricing
* Team members
* Contact information
* Business hours
* Testimonials

---

# Styling Changes

Colors, spacing, typography, and layout settings are located in:

style.css

The primary color variables are located near the top of the file.

---

# JavaScript Functionality

Interactive functionality is contained inside:

script.js

Features include:

* Mobile navigation
* Theme switching
* Scroll animations
* Form interactions
* Smooth scrolling

---

# Replace Placeholder Content

Before deploying your website, remember to:

* Replace images
* Update phone numbers
* Update email addresses
* Update social media links
* Configure Formspree
* Test all links
